# 文档

本项目的文档已经迁移至 doc 分支。  
您可以前往 tmoe 的 doc 网站在线阅览。

[doc.tmoe.me](https://doc.tmoe.me)

您如果无法访问此网站，那就手动克隆 **doc** 或 **gh-pages** 分支吧！

```sh
git clone \
    -b doc \
    --depth=1 \
    https://github.com/2moe/tmoe
```

或者是

```sh
git clone \
    -b gh-pages \
    --depth=1 \
    https://github.com/2moe/tmoe \
    pages

xdg-open pages/zh/index.html
```
